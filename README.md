# agentic-python-dependency

Minimal starter scaffold for a Python repository.

## Requirements

- Python 3.12+

## Quick start

```bash
pip install -e .
agentic-python-dependency
```

## Commit convention

This repository uses Conventional Commits via Commitizen.

Examples:

- `feat: add initial project scaffold`
- `fix: correct workflow trigger`
- `docs: expand setup instructions`
- `chore: update dev tooling`

## Changelog

Changes are tracked in [CHANGELOG.md](CHANGELOG.md). Release automation is intentionally manual for now.
